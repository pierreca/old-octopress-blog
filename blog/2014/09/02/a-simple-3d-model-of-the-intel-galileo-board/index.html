A Simple 3D Model of the Intel Galileo Board

Sep 2nd, 2014

I published a very simplistic 3d model of Intel’s Galileo (v1) board on Github 2 days ago. It was built using OpenScad and the code is available as well as the exported STL file.

I rely on 3D models a lot during the design of any project because it’s a way for me to cut on prototyping costs and times. One thing that often leaves me wanting more with a lot of projects I see from makers is the fact that it stays on a breadboard. There’s rarely any PCB design, and even more rarely enclosure or industrial design that would make the thing sturdy and reliable enough to function on something else than a desk… I guess I’ll write about that and maybe some techniques that I have found useful later…

Please try this design with various enclosures, use it, and let me know if it works well or if something’s off!
